# 공모전 논리사슬 확정법

## 목적

논리사슬은 아이디어의 구성요소 목록이 아니다. 심사자가 `이 제안을 선택해야 한다`는 결론에 도달하도록 각 판단을 필연적으로 연결한 증명 구조다. 원고를 쓰기 전에 사슬을 확정하고, 모든 문단을 사슬의 한 고리를 수행하게 하라.

## 두 개의 사슬을 하나로 결합하기

공모전 원고에는 두 사슬이 동시에 필요하다.

### 1. 선택 논리사슬

`채택 결론 → 공모 목적 → 문제·시급성 → 대상 → 원인 → 기존 공백 → 개입 → 비교우위 → 실행 증명 → 평가 판정 → 채택 결론`

이 사슬은 `왜 이 제안을 선택해야 하는가`를 증명한다.

### 2. 변화 논리사슬

`투입 → 활동 → 직접 산출 → 중간 변화 → 최종 성과 → 측정 지표`

이 사슬은 `제안이 어떻게 실제 변화를 만드는가`를 증명한다.

두 사슬은 `개입 → 활동`과 `성과 → 공모 목적`에서 결합해야 한다. 선택 논리만 있으면 구호가 되고, 변화 논리만 있으면 운영계획이 된다.

| --- | --- | --- | --- | --- | --- | --- |

| 4. 구조적 원인 | 반복되는 문제를 만드는 핵심 원인은 무엇인가 |  |  |  |  |  |

## 각 고리 작성 공식

각 고리를 다음 문장군으로 완성하라.

1. `명제`: 심사자가 받아들여야 할 결론을 단정한다.
2. `근거`: 그 결론을 받치는 자료·관찰·비교·설계·실행계획을 제시한다.
3. `논거`: 그 근거가 왜 이 결론을 증명하는지 설명한다.
4. `전환`: 그래서 다음에 어떤 판단이 필요한지 연다.

예시 구조:

> `[근거]`는 `[확정 명제]`를 보여준다. 핵심은 `[근거와 명제를 잇는 논거]`다. 따라서 해결책은 `[다음 고리의 선택 기준]`을 충족해야 한다.

근거만 있고 논거가 없으면 심사자가 대신 추론해야 한다. 논거만 있고 근거가 없으면 주장이다. 둘을 한 쌍으로 묶어라.

## 논리 점프 탐지

다음 패턴을 찾으면 고리를 복구하라.

- 문제 규모 다음에 곧바로 기능 목록이 나옴
- 현상과 원인을 같은 말로 처리함
- 개입이 원인이 아니라 문제의 표면만 건드림
- 활동을 성과로 부름
- 산출 다음에 중간 변화 없이 장기효과로 점프함
- 비교 기준이 기존 대안과 제안에서 달라짐
- 실행 의지를 실행 증거로 사용함
- 공모 목적과 기대효과가 단어만 비슷하고 인과 연결이 없음
- 통계가 문제 존재만 증명하는데 제안 효과까지 주장함
- 결론에서 본문에 없던 가치를 새로 추가함

점프를 발견하면 접속어를 추가하지 마라. 빠진 원인, 중간 변화, 비교 기준, 실행 조건, 측정 지표 중 필요한 고리를 실제 내용으로 채워라.

## 정방향 검사

다음 질문에 모두 한 문장으로 답하라.

1. 문제는 어떤 원인 때문에 반복되는가
2. 제안은 그 원인의 어떤 지점을 바꾸는가
3. 그 변화는 사용자의 어떤 행동 또는 절차를 바꾸는가
4. 바뀐 행동은 어떤 직접 산출을 만드는가
5. 그 산출은 어떤 중간 변화를 만드는가
6. 중간 변화는 어떤 성과지표에 반영되는가
7. 그 성과는 공모 목적을 어떻게 달성하는가

하나라도 답이 없으면 해당 연결을 먼저 설계하라.

## 역방향 검사

최종 성과에서 거꾸로 추적하라.

`목표 성과를 만들려면 어떤 중간 변화가 필요한가 → 그 변화를 만들 산출은 무엇인가 → 산출을 만들 활동은 무엇인가 → 활동을 수행할 주체·자원·권한은 무엇인가`

역추적에서 새 요소가 나오면 실행계획에 추가하고, 정방향 사슬에도 같은 용어로 반영하라.

비교는 같은 기준에서 수행하라.

| --- | --- | --- | --- |
| 대상 적합성 |  |  |  |
| 문제 원인 타격 |  |  |  |
| 절차·시간·비용 |  |  |  |
| 실행 가능성 |  |  |  |
| 측정 가능성 |  |  |  |
| 확산 가능성 |  |  |  |

모든 기준을 채우지 말고 실제 평가항목과 제안의 승부처에 해당하는 2~4개만 선택하라. `더 편리하다`, `더 혁신적이다` 대신 차이가 발생하는 단계와 결과를 써라.

## 득점 연결

논리사슬의 각 고리는 적어도 하나의 공식 평가항목에 연결되어야 한다. 높은 배점 항목은 여러 고리에서 서로 다른 증거로 강화하라.

| 공식 항목 | 최고점 판정 문장 | 논리사슬 고리 | 핵심 증거 | 심사자가 인용할 문장 |
| --- | --- | --- | --- | --- |
|  |  |  |  |  |

공식 항목에 연결되지 않는 문장은 삭제 후보로 분류하라. 다만 다음 고리를 이해시키는 짧은 전환문은 유지할 수 있다.

## 사슬 확정 판정

다음을 모두 통과하면 `확정`이다.

- 채택명제가 하나다.
- 모든 고리에 명제와 증거가 있다.
- 증거와 명제 사이의 논거가 명시되어 있다.
- 각 고리가 다음 고리를 필연적으로 연다.
- 개입이 구조적 원인을 직접 바꾼다.
- 활동·산출·성과가 구분된다.
- 성과에 지표·측정 방법·시점이 있다.
- 비교우위가 같은 기준에서 증명된다.
- 실행주체·자원·일정·완료 기준이 있다.
- 모든 공식 평가항목이 하나 이상의 고리에서 회수된다.
- 마지막 결론이 첫 채택명제를 더 강하게 회수한다.
